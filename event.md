# [アクセシビリティキャンプ東京 #8](http://atnd.org/event/a11ytokyo8/)	{#camp-8}

開催日時
:   2013年11月1日（金）19:00〜22:30

場所
:   株式会社ミツエーリンクス 本社 セミナールーム

第8回では「W3C/WAI EasyCheck ～自分でアクセシビリティをチェックしてみよう～」と題し、実際にアクセシビリティチェックの第一歩を体験するワークショップを実施しました。
初心者の方を中心とした5名程度のグループ4つでは[W3C WAIが提供しようとしているEasy Checksの草案](http://www.w3.org/WAI/eval/preliminary.html)の項目をベースに、実際のツールの使い方やチェックの観点、あるべき姿などについて勉強し、疑問点等について議論しました。
別途アクセシビリティの専門家チームも結成したところ、こちらではWAI-ARIAの適用方法など、最近のWeb表現に対するアクセシビリティ確保の方法などに関する深い議論がなされました。

### 当日の様子

* [togetterまとめ](http://togetter.com/li/584538)
